# 🧊 Cube Math — 用线性代数思维还原魔方

**魔方不是靠背的，是用数学推出来的。** 一个通过线性代数与群论概念理解魔方变换、自主推导复原方案的教学网站。

🔗 **在线地址**：[https://tommao46.github.io/cube-math/](https://tommao46.github.io/cube-math/)

---

## 📖 项目概述

Cube Math 是一个交互式魔方教学工具，将经典的**7步层先法复原**与**线性代数/群论**概念一一对应：

| 步骤 | 魔方操作 | 数学概念 |
|:---:|------|----------|
| 1 | 白色十字 | 方向指数坐标系 |
| 2 | 白色角块 — R' D' R D | 逆序列法则 |
| 3 | 中层棱块 — U R U' R' U' F' U F | 共轭变换 |
| 4 | 黄色十字 — F R U R' U' F' | 交换子 |
| 5 | 棱块归位 — R U' R U R U R U' R' U' R² | 置换群 |
| 6 | 角块归位 — U R U' L' U R' U' L | 交换子 / 奇置换 |
| 7 | 翻黄角 — R U R' U R U² R' | Sune / 旋转矩阵 |

每一步都遵循 **介绍 → 引导演示 → 亲手实操** 的递进式教学，并配有矩阵解读和交互式求解演示面板。

## ✨ 功能特性

- **3D 交互魔方** — 基于 Three.js + React 的实时 3D 渲染，支持鼠标点击旋转、右键/键盘操作
- **首页自动演示** — 打乱 → 还原循环动画，每步显示公式标注
- **7 步引导复原** — 介绍 + 逐步动画 + 实操检测，含步骤快捷跳转栏
- **交互式求解面板** — 第 6 节集成 SolveDemoPanel，7 步解法可逐步点播
- **交换子动画** — 独立 CommutatorDemo 组件，可视化 "打开→操作→关上" 结构
- **实操训练** — 教学模式（矩阵解读）、练习模式（提示）、挑战模式（无提示）
- **键盘快捷键** — RLUDFB/方向键 = 转动，Shift = 逆时针，Ctrl+Z = 撤销，Space = 打乱
- **6 章理论知识** — 状态向量、旋转矩阵、交换律验证、逆矩阵、不变量、复原实战
- **进阶档案馆** — 类比边界、置换群、深度数学结论、拓展资源（折叠动画）
- **GitHub Pages 部署** — 静态站点，零后端依赖

## 📐 数学概念对应

| 线代/群论概念 | 魔方中的体现 |
|------|------|
| **方向指数坐标系** | 6 面颜色 = 6 个基准方向，每块有位置 + 朝向两个属性 |
| **旋转矩阵（特殊正交群 SO(3)）** | 每次转动 = 绕 X/Y/Z 轴 90°，是正交矩阵且行列式为 1 |
| **矩阵乘法不满足交换律** | R·U ≠ U·R，顺序影响结果 |
| **逆矩阵** | 逆时针 = 顺时针矩阵的转置（A⁻¹ = Aᵀ） |
| **逆序列法则** | (A·B)⁻¹ = B⁻¹·A⁻¹，后做的先撤销 |
| **共轭变换** | A·X·A⁻¹ = 在 A 变换后的坐标系中执行 X |
| **交换子** | A·B·A⁻¹·B⁻¹ = 只改变目标区域，不影响其他位置 |
| **置换群** | 每次转动 = 贴纸位置的置换，符合偶置换约束 |
| **Sune（3 循环）** | 只翻转角块朝向，位置不变 |
| **乘法不可交换** | Sune 的效果与执行顺序强相关 |
